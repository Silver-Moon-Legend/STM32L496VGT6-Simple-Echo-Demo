# 硬件接线指南

## 引脚配置

### STM32L496VGT6引脚映射
```
PC2 → DFSDM1_CKIN0 (麦克风时钟)
PC3 → DFSDM1_DATIN0 (麦克风数据)
PB12 → I2S3_WS (MAX98357 LRCLK)
PB15 → I2S3_SD (MAX98357 DATA)
PC10 → I2S3_SCK (MAX98357 SCLK)
PC12 → I2S3_MCK (MAX98357 MCLK)
```

### PCB布局建议
1. **电源部分**：3.3V电源滤波
2. **麦克风部分**：远离喇叭以避免噪声干扰
3. **喇叭部分**：喇叭位置远离麦克风
4. **STM32部分**：中心位置，便于连接
